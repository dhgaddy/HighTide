Design Gallery

Placement views, upstream links, and QoR metrics for each design in the HighTide benchmark suite.

BlackParrot

SystemVerilog asap7 nangate45
Die image pending

BlackParrot is a Linux-capable RISC-V multicore processor from the University of Washington. HighTide includes two variants: bp_uno (single unicore) and bp_quad (quad-core). The design uses dense SRAM macros for caches and register files.

PPA metrics will be populated once builds complete.

Gemmini

Chisel asap7 nangate45

Gemmini is a systolic array matrix multiplication accelerator from UC Berkeley, generated from Chisel. It is the largest macro-free design in the suite (~570k cells), testing how the flow scales with large, regular datapath structures.

PPA metrics will be populated once builds complete.

SHA3

Chisel asap7 nangate45

SHA3 is a cryptographic hash accelerator from UC Berkeley implementing the Keccak/SHA-3 algorithm. A small, macro-free design (~15k cells) dominated by combinational logic, exercising synthesis optimization and logic density.

PPA metrics will be populated once builds complete.

NyuziProcessor

NyuziProcessor is a GPGPU-style multicore vector processor with hardware multithreading, vector arithmetic, and an L1/L2 cache hierarchy. Uses multiple SRAM macros for caches and register files across its cores.

An LFSR-based pseudo-random bit sequence generator. The smallest design in the suite, available on all three platforms. Useful for rapid flow validation and as a baseline for PPA optimization experiments.

LiteEth

Migen/LiteX asap7 nangate45 sky130hd
Die image pending

LiteEth is a lightweight Ethernet core generated from Python via the LiteX/Migen framework. HighTide includes six MAC/PHY variants (MII, RGMII, SGMII) covering different interface configurations, each with embedded SRAM macros. Available on all three platforms.

PPA metrics will be populated once builds complete.

FlooNoC is a network-on-chip from the PULP Platform group at ETH Zurich, configured as a 4x4 mesh of black-box cluster endpoints. Read directly as SystemVerilog via yosys-slang. Uses the Bender dependency manager and FloGen for mesh topology generation.

Snitch Cluster is an energy-efficient compute cluster from the PULP Platform group at ETH Zurich. Uses the Bender dependency manager and clustergen for cluster wrapper generation. Read directly as SystemVerilog via yosys-slang with hierarchical synthesis and embedded SRAM macros.
